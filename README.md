# Yapay Zeka Destekli Online Sınav ve Değerlendirme Sistemi 🚀

Bu proje, öğrencilerin online olarak sınavlara girmesini, anlık değerlendirme almasını ve yöneticilerin sınavları kolayca yönetebilmesini sağlayan bir **Full-Stack Web Uygulamasıdır.**

## 🛠 Kullanılan Teknolojiler
* **Backend:** PHP (OOP & PDO)
* **Veritabanı:** MySQL (İlişkisel Veritabanı Mimarisi)
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Dinamik Etkileşim:** JavaScript (Live Search, DOM Manipulation)

## 🌟 Öne Çıkan Özellikler
* **Canlı Arama (Live Search):** Sayfa yenilenmeden, veritabanını yormadan anlık sınav filtreleme.
* **İlişkisel Veritabanı:** `categories` ve `exams` tabloları Foreign Key ile bağlanarak veri tekrarı önlendi.
* **Sosyal Kanıt (Social Proof):** "Bu sınavı 12 kişi çözdü" gibi anlık veriler SQL Subquery'leri ile hesaplanır.
* **Güvenlik:** PDO Prepared Statements ile SQL Injection koruması.
